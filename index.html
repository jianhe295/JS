Learn JavaScript

You can run JavaScript code with only a text editor and a browser. You can use any text editor in Linux or Notepad in Windows.
Later on you can learn how to use code editor such as Sublime Text. Below are the steps to run the code.

1. I use Text Editor in Ubuntu 24.01 and Firefox. First, open Text Editor to enter the codes below and save it as myJS1.html.
You may copy and paste. 


<!DOCTYPE html>
<title>LEARN JAVASCRIPT</title>
Press F12, click on console if it is not selected.
   

<script>


let num1 = 6;
let num2 = 3;


let result = num1 + num2;


console.log('The answer is: ' + result)

</script>

2. After you save it, go to Files/File Manager or File Explorer to search for myJS1.html. Right-click and choose Open with Firefox.
The browser will open. Press F12 to open console. Now, you can see the answer.

3. Go back to Text Editor and open the file myJs1.html. Change num1 and num2, you also can change + to -. Save the file and open it in browser gain.
Press F12 to see the new answer.
